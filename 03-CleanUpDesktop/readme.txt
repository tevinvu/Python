# This Python script will help you to clean up your messy desktop.
# This one works for Mac only
It will move all of your PDF files on your desktop to a folder name PDF
It will move all of your pictures with .jpg extension to a folder name Pictures
It will move all of your PowerPoint files to a folder name PowerPoints
It also will move all of your mp3 files to a folder name mp3
Download: use GitHub to download 03-CleanUpDesktop.py
It will be automatic find your desktop path and clean it up for you.
Run: python 03-CleanUpDesktop.py

Hope this python script automation will help you keep your desktop clean.
